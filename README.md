# ARE YOU READY? - Tactical Special Forces

A top-down tactical shooter built with Pygame.

## Download

Grab the latest Windows EXE builds from the [Releases](https://github.com/vfyov6621-coder/are-you-ready/releases) page.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Aim |
| Left Click | Shoot |
| E | Interact (open/close doors) |
| ESC | Pause |

## Running from Source

```bash
pip install -r requirements.txt
cd game
python main.py
```

## Map Constructor

```bash
cd constructor
python main.py
```
